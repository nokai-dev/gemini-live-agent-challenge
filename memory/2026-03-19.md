### Key Discovery: DevPost API

DevPost has a public API at `https://devpost.com/api/hackathons`:
- `?filter=ended&managed_by_devpost_badge=1` — all past official hackathons
- `?managed_by_devpost_badge=1` — all time (open + closed)
- **147 total DevPost-managed hackathons** exist (not just Gemini)
- Returns: title, organization_name, registration_count, open_state, themes, prize_amount, winners_announced, url, submission_gallery_url

## Daily Hackathon Checker (check_new_hackathons.py)

Location: `/data/.openclaw/workspace/gemini-history/check_new_hackathons.py`

**What it does:**
1. Calls `https://devpost.com/api/hackathons?managed_by_devpost_badge=1`
2. Filters for Google/Gemini/AI-related via keyword matching
3. Compares against previously seen hackathons (state file)
4. Sends Telegram alert for new worth-it hackathons
5. Logs to `checker.log`

**Keywords that trigger alerts:** gemini, google ai, google cloud, google deepmind, chrome built-in, vertex ai, generative ai, gen ai, llm, multimodal ai, ai agent, live agent, agentic, tensorflow

**Blocklist:** mlh local, student, campus, university, workshop

**Still needs:** TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID env vars to be set
