### Voice Commands (Speak Clearly)

1. **"Make this button blue"** — 0:14
2. **"Add more padding here"** — 0:29
3. **"Make this font bigger and bolder"** — 0:44
4. **"Make these cards a grid layout, three columns"** — 0:59

### Timing Tips

- **Pause 1 second** after each command for AI processing
- **Pause 2 seconds** after each visual change to let it sink in
- **Speak confidently** — this is the future, own it
- **Keep cursor movements smooth** — no jerky movements

### Backup Scenarios

| If This Happens | Do This |
|-----------------|---------|
| Gemini API slow | Cut to "Thinking..." → pre-recorded result |
| Component misdetected | Retry once, then use backup recording |
| File write fails | Show diff view only, explain "apply" step |
| Hot reload fails | Manual refresh, keep talking |
